# 제품 요구사항 정의서 (PRD) - SellFlow-AI
**작성일**: 2025-11-23
**버전**: 1.2 (Vision Refined)

## 🟢 WHY | 왜 만드는가

### 💡 배경과 문제
*   **1인 셀러의 한계**: 혼자서 소싱, 분석, 상세페이지, 디자인, 업로드까지 하느라 하루 8시간 이상 소요.
*   **전문성 부족**: 디자인(포토샵), 카피라이팅, 법적 지식(상표권) 부족으로 경쟁력 저하.
*   **단순 반복**: 창의적인 전략보다 단순 등록 노가다에 시간 낭비.

### 🚦 변화와 기회 (Vision)
*   **"1인 이커머스 사업자를 위한 완전 자동화 솔루션"**: AI가 단순 반복 업무를 100% 처리.
*   **HITL (Human-in-the-Loop) 시너지**: AI가 초안을 만들고, **셀러의 센스(감각)**로 화룡점정. (예: "이 문구는 좀 더 감성적으로", "배경을 카페로 바꿔줘")
*   **법적 안전장치**: **나노바나나(NanoBanana)** 등 생성형 AI를 활용해 기존 이미지를 변형(배경/모델 변경)하여 저작권 문제 원천 차단.

### 🎯 목표
1.  **완전 자동화**: 소싱부터 업로드 대기까지 원스톱 처리.
2.  **법적 리스크 Zero**: AI 재가공을 통한 이미지/상표권 안전 확보.
3.  **사용자 주도권**: AI는 도구일 뿐, 최종 결정과 퀄리티 컨트롤은 사용자가 수행.

## 🟠 WHAT | 무엇을 만드는가

### 👤 사용자 페르소나
| 이름 | 직업 | 니즈 (Needs) | 솔루션 (Solution) |
| :--- | :--- | :--- | :--- |
| **김철수** | 1인 스마트스토어 셀러 | "내 센스를 더해서 퀄리티를 높이고 싶어." | AI가 만든 초안에 자연어로 피드백("키워드 빼줘", "배경 바꿔")하여 즉시 수정. |

### 🌟 핵심 기능 및 흐름
1.  **경쟁사 분석**: 잘 팔리는 경쟁사 페이지 URL 입력 → AI가 소구점/키워드 추출.
2.  **콘텐츠 생성 (Creative)**:
    *   **텍스트**: 추출된 소구점으로 상세페이지/카피 작성.
    *   **이미지 (NanoBanana)**: 제품 누끼는 살리고, 배경/모델을 생성하여 **"세상에 없는 안전한 이미지"** 제작.
3.  **HITL 피드백**: 웹 대시보드에서 결과물 확인 및 수정 명령.
4.  **자동 업로드**: 최종 승인 시 **n8n**이 오픈마켓 API를 통해 자동 등록.

### 🌟 핵심 기능별 (에이전트별) 상세 테이블
| 기능 | 역할/자동화 | 셀러 개입 (HITL) | 체크리스트 (완료 조건) |
| :--- | :--- | :--- | :--- |
| **상품 소싱 자동화** | 인기/트렌드상품, 데이터 기반 매일 추천 | 카테고리/필터/키워드 지정 | 일 50개 이상 자동 발굴 |
| **경쟁사 분석** | 광고/상품 인사이트 자동 도출 | 전략키워드/광고 직접 선택 | 주간 20개 이상 분석 |
| **상세페이지/카피 생성** | 콘셉트/문구/이미지 생성, 맞춤 카피 | 문구·이미지 개별 피드백·명령 | 3가지 변형·A/B테스트 생성 |
| **이미지 안전화** | 상품 사진 AI로 배경/연출 변경 | 조건 지정 (색, 스타일 등) | 법적 리스크 없는 이미지 |
| **자동 업로드/관리** | 승인 후 플랫폼 일괄 등록, 관리 리포트 | 승인/반려/재작업 요청 | 95% 자동 / 5% 수작업 관리 |

### 🏆 성공 기준
*   5개 에이전트 안정 협업, 자연어 명령·업로드 자동화
*   전체 작업의 **80% 이상 자동 처리**
*   사용자 피드백 기반 **품질 90점 이상 자동 승인**
*   법적 이슈 **0건** (상표권·이미지)
