## 🎯 사용자 플로우 (User Journey)

### Flow 1: 신규 사용자 온보딩

```
Landing Page → 회원가입 → API 키 설정 → 첫 번째 상품 소싱 → 대시보드
```

### Flow 2: 상품 등록 자동화

```
대시보드 → 소싱 (키워드 입력) → 상품 선택 → 경쟁사 분석 → 
콘텐츠 생성 → HITL 피드백 → 승인 → 자동 업로드 → 완료 알림
```

### Flow 3: 기존 상품 관리

```
대시보드 → 상품 관리 → 상품 선택 → 수정/삭제/재업로드
```
